# Components

The shared UI package provides a comprehensive set of reusable React components built with modern web standards.

## Overview

All components are built with:
- **TypeScript** for type safety
- **Tailwind CSS** for styling
- **Radix UI** primitives for accessibility
- **Class Variance Authority** for variant management

## Available Components

### Button

A versatile button component with multiple variants and sizes.

```tsx
import { Button } from '@repo/shared-ui';

<Button variant="default">Default</Button>
<Button variant="destructive">Destructive</Button>
<Button variant="outline">Outline</Button>
<Button variant="secondary">Secondary</Button>
<Button variant="ghost">Ghost</Button>
<Button variant="link">Link</Button>
```

### Card

A flexible card component for displaying content.

```tsx
import { Card, CardHeader, CardTitle, CardDescription, CardContent, CardFooter } from '@repo/shared-ui';

<Card>
  <CardHeader>
    <CardTitle>Card Title</CardTitle>
    <CardDescription>Card description</CardDescription>
  </CardHeader>
  <CardContent>
    <p>Card content goes here</p>
  </CardContent>
  <CardFooter>
    <p>Card footer</p>
  </CardFooter>
</Card>
```

### Input

A styled input component for forms.

```tsx
import { Input } from '@repo/shared-ui';

<Input type="text" placeholder="Enter text" />
<Input type="email" placeholder="Enter email" />
<Input type="password" placeholder="Enter password" />
```

## Usage

To use these components in your app:

1. Install the shared UI package:
   ```bash
   npm install @repo/shared-ui
   ```

2. Import and use components:
   ```tsx
   import { Button, Card, Input } from '@repo/shared-ui';
   ```

3. Make sure Tailwind CSS is configured in your app.

## Customization

Components can be customized using:
- CSS variables for theming
- Tailwind CSS classes
- Component variants
- Custom props
